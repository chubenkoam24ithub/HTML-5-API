## Описание проекта

**"HTML5 API Demo"** — это современное одностраничное веб-приложение (SPA), демонстрирующее мощь **HTML5 API** без использования фреймворков. Проект создан на чистом **JavaScript**, **HTML5** и **CSS3** и включает:

- **History API** — навигация по страницам без перезагрузки  
- **Canvas API** — интерактивная анимация шариков с физикой  
- **Web Workers** — фоновые вычисления с прогрессом и отменой  
- **LocalStorage** — сохранение состояния Canvas  
- **Service Worker** — оффлайн-доступ и PWA-функции  
- **Notification API** — уведомления о переходах  
- **Тёмная тема** с `prefers-color-scheme`  

Цель — показать, как создавать **производительные, отзывчивые и оффлайн-доступные** веб-приложения с использованием только **встроенных браузерных API**.

## Функционал

| Раздел | Описание |
|-------|--------|
| **Home** | Приветственная страница с навигацией |
| **About** | Описание технологий и особенностей проекта |
| **Canvas** | Интерактивный холст: клик → добавление шарика, гравитация, отскоки, FPS-счётчик |
| **Worker** | Web Worker: вычисление суммы квадратов до 10M+ с прогресс-баром и кнопкой отмены |

> Все состояния (шарики, анимация) сохраняются в `localStorage` каждые 5 секунд.

## Используемые HTML5 API

| API | Применение |
|-----|----------|
| **History API** | SPA-роутинг (`pushState`, `popstate`) |
| **Canvas API** | Анимация шариков, `requestAnimationFrame`, FPS |
| **Web Workers** | Асинхронные вычисления, прогресс, отмена |
| **LocalStorage** | Сохранение Canvas-состояния |
| **Service Worker** | Кэширование, оффлайн-доступ |
| **Notification API** | Уведомления при переходе по страницам |
| **prefers-color-scheme** | Автоматическая тёмная тема |
